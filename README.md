## Установка и запуск проекта

Клонировать репозиторий:

`git clone git@github.com:Bambyary/movies-explorer-api.git`

Установить зависимости:

`npm i`

Запустить сервер:

`npm run start`

Запустить сервер с hot-reload:

`npm run dev`
